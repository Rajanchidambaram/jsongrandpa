# JSON Grandpa

JSON Grandpa is a lightweight, privacy-first JSON Validator & Formatter built with Next.js. It validates, formats and compresses JSON entirely in the browser.

## Features
- Validate JSON with error messages
- Pretty-print and minify
- Monaco editor UI
- Keyboard shortcuts
